# webpack-boilerplate

webpack boilterplate 입니다.<br><br> eslint-config-airbnb 가 적용되어있습니다. <br><br> 다운로드 혹은 git clone 하여 `npm install` 실행 후 사용하시면 됩니다.


## Installation

```bash
> npm install
```


## Run Dev Server

```bash
> npm run dev
```

## Build

```bash
> npm run build
```
